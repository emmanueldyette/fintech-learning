## Program #1 - budget_tracker.py

## What The Program Does:
- Helps users see exactly how their income is spent on a monthly basis through a detailed breakdown of each expense and the percentage of income that it takes up.
- Gives advice on the user's saving habits and specific expenses

## Why I Built this Program:
- In this day and age, the need for a personal budget has increasingly grown. With rising food and housing costs and little to no increase in income, budgeting is a much needed method that can be used to cut back on expenses

## How to Run It:
- Make sure Python 3 is installed, then: python budget_tracker.py

## Technologies
- Python 3
- No external libraries required
